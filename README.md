# Projekt1_robotdammsugare
TP projekt 1. Robotdammsugare som styrs genom raspberry pi zero som använder open cv för att detektera skräp och navigera till de. Projket körs på både en raspberry pi zero som är hjärnan och även arduino som styr motorerna och tar emot data från sensorerna. Och det finns möjlighet att starta och stänga av dammsuagren via en hemsida.
